# Number Wizard

## Beschreibung
Number Wizard ist ein einfaches Unity-Ratespiel.
Das Spiel versucht, die vom Spieler gedachte Zahl zu erraten.

## Bedienung
- **Higher** – die gesuchte Zahl ist höher als der aktuelle Vorschlag
- **Lower** – die gesuchte Zahl ist niedriger als der aktuelle Vorschlag
- **Correct** – der aktuelle Vorschlag ist richtig
- **Restart** – startet das Spiel neu

## Funktionsweise
Das Spiel verwendet binäre Suche.
Nach jedem Hinweis wird der Suchbereich halbiert und ein neuer Guess berechnet.
